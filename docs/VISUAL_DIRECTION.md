# VISUAL DIRECTION — CLARICE MARGON CONSÓRCIOS

## OBJETIVO DESTE DOCUMENTO

Este arquivo define a direção visual e artística da nova versão da landing page de Clarice Margon.

A primeira versão criada pelo Codex ficou visualmente genérica, sem força de composição, sem ritmo comercial e distante da referência desejada.

Este documento existe para corrigir isso.

A função dele é orientar o Codex sobre COMO a página deve parecer visualmente, não apenas quais seções ela deve ter.

A landing page deve ter aparência:

- premium;
- comercial;
- elegante;
- sofisticada;
- feminina com autoridade;
- clara;
- confiável;
- bem diagramada;
- com conteúdo concentrado no miolo;
- com ritmo visual parecido com a referência do Ezequiel;
- com identidade própria da Clarice Margon.

--------------------------------------------------

# 1. PROBLEMA DA PRIMEIRA VERSÃO

A primeira versão da landing page foi rejeitada porque ficou:

- genérica;
- com aparência de template;
- com Hero sem impacto;
- com copy fraca;
- com H1 quebrando feio;
- com seções soltas;
- com pouco ritmo visual;
- com mapa abstrato sem leitura de países;
- com gráfico de linha fraco;
- com pré-simulação desnecessária;
- com apresentação da Clarice cedo demais;
- com estética distante da referência desejada.

A nova versão não deve tentar apenas corrigir pequenos detalhes.

A nova versão deve reconstruir a página com outra direção visual.
