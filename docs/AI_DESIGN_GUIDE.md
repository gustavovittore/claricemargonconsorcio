# AI DESIGN GUIDE — CLARICE MARGON CONSÓRCIOS

## OBJETIVO DESTE DOCUMENTO

Este arquivo orienta a direção visual da landing page de Clarice Margon.

A landing page deve parecer premium, elegante, confiável e estratégica.

O objetivo NÃO é criar uma página genérica de venda de consórcio.

O objetivo é posicionar Clarice Margon como uma consultora financeira especializada em consórcios, planejamento patrimonial e construção de patrimônio.

A página deve transmitir:

- autoridade;
- sofisticação;
- confiança;
- clareza;
- segurança;
- elegância;
- planejamento;
- inteligência financeira;
- atendimento humano e estratégico.

--------------------------------------------------

# POSICIONAMENTO VISUAL

A landing page deve parecer mais próxima de:

- consultoria patrimonial premium;
- assessoria financeira sofisticada;
- wealth management;
- private banking;
- imobiliário de alto padrão;
- consultoria estratégica.

E NÃO deve parecer:

- panfleto de consórcio;
- site genérico de banco;
- página agressiva de tráfego pago;
- layout de vendedor comum;
- página poluída;
- fintech barata;
- template pronto sem personalidade.

--------------------------------------------------

# IDENTIDADE VISUAL

## Paleta principal oficial

A identidade visual da landing page deve seguir esta paleta:

- Cor principal de fundo: `#F2F0EF`
- Cor principal dos textos: `#1F1F1F
- Cor de destaque, botões e detalhes: `#E19D35`

A base da página deve ser predominantemente off-white, com textos em preto e detalhes estratégicos em dourado.

O dourado deve ser usado com sofisticação, principalmente em:

- botões principais;
- ícones;
- bordas finas;
- números importantes;
- pequenos detalhes visuais;
- destaques de seções;
- elementos premium.

Não usar o dourado em excesso.
O dourado deve parecer elegante, não chamativo.

## Sensação da paleta

A paleta deve transmitir:

- luxo discreto;
- estabilidade;
- confiança;
- patrimônio;
- maturidade;
- organização financeira.
--------------------------------------------------

# TIPOGRAFIA

A tipografia deve ser:

- sofisticada;
- limpa;
- moderna;
- fácil de ler;
- com hierarquia forte.

## Títulos

Os títulos devem ter aparência premium, com peso visual forte, mas sem exagero.

Características:

- grandes;
- elegantes;
- com bom espaçamento;
- alinhamento bem pensado;
- que transmitam autoridade.

## Textos

Os textos devem ser:

- claros;
- diretos;
- humanos;
- consultivos;
- fáceis de ler;
- sem blocos longos demais.

Evitar textos gigantes sem quebra visual.

--------------------------------------------------

# ESTILO GERAL

A página deve ter visual:

- premium;
- clean;
- elegante;
- moderno;
- financeiro;
- confiável;
- estratégico;
- sofisticado;
- com boa respiração visual.

O layout deve usar bastante espaço em branco.

Nada deve parecer apertado.

Os elementos devem parecer intencionais, refinados e bem distribuídos.

--------------------------------------------------

# HERO SECTION

A primeira dobra da página deve ser extremamente forte.

Ela precisa comunicar imediatamente:

- quem é Clarice Margon;
- o que ela faz;
- por que ela é confiável;
- para quem ela atende;
- qual transformação ela oferece;
- qual o próximo passo do visitante.

## Sensação da hero

A hero deve parecer:

- premium;
- segura;
- consultiva;
- elegante;
- estratégica;
- com autoridade.

## Evitar

Não criar uma hero genérica com apenas texto e botão.

Não criar uma hero com aparência de template.

Não usar excesso de elementos.

Não usar visual muito bancário ou frio.

## Elementos recomendados

A hero pode conter:

- foto profissional da Clarice;
- composição premium com fundo gelo/off-white;
- card flutuante com número ou benefício;
- selo de credibilidade;
- menção a bancos parceiros;
- CTA para simulação no WhatsApp;
- microelementos dourados;
- animações suaves de entrada.

--------------------------------------------------

# IMAGENS

As imagens devem reforçar:

- confiança;
- patrimônio;
- futuro;
- família;
- imóveis;
- planejamento;
- autoridade da Clarice.

Se usar foto da Clarice, ela deve aparecer de forma elegante e profissional.

Se usar imagens complementares, preferir:

- imóveis sofisticados;
- famílias bem posicionadas;
- brasileiros pensando no futuro;
- ambientes claros e premium;
- cenas com aparência editorial.

Evitar:

- imagens genéricas demais;
- fotos com aparência artificial;
- fotos exageradamente comerciais;
- pessoas apontando para botões;
- banco de imagem barato.

--------------------------------------------------

# CARDS

Os cards devem ter aparência premium.

Características:

- fundo branco ou off-white;
- bordas suaves;
- sombras leves;
- cantos arredondados;
- espaçamento interno confortável;
- ícones elegantes;
- textos curtos;
- hierarquia clara.

Não usar cards apertados.

Não usar sombras pesadas.

Não usar excesso de ícones.

--------------------------------------------------

# BOTÕES

Os botões devem parecer sofisticados e importantes.

## Botão principal

O CTA principal deve ser para WhatsApp.

Exemplo:

- Fazer minha simulação
- Simular meu consórcio
- Falar com a Clarice
- Receber análise personalizada
- Quero construir patrimônio

## Estilo

O botão principal pode usar:

- dourado elegante;
- texto escuro;
- borda refinada;
- hover suave;
- leve animação;
- presença visual forte.

## Botões secundários

Podem ser mais discretos:

- fundo transparente;
- borda fina;
- texto escuro;
- hover refinado.

--------------------------------------------------

# SEÇÕES IMPORTANTES

A landing page deve conter seções estratégicas como:

1. Hero section premium;
2. Logos ou menção aos bancos parceiros;
3. Seção de autoridade da Clarice;
4. Seção sobre consórcio como estratégia patrimonial;
5. Seção focada em brasileiros no exterior;
6. Benefícios do consórcio;
7. Comparativo entre financiamento e consórcio;
8. Números animados;
9. Gráfico ou visual educativo;
10. Como funciona o atendimento;
11. Tipos de consórcio atendidos;
12. Depoimentos ou prova social em vídeo, se houver;
13. FAQ;
14. CTA final para WhatsApp.

--------------------------------------------------

# BANCOS PARCEIROS

Clarice trabalha com acesso a grandes instituições e administradoras.

Usar essa informação como prova de autoridade.

Bancos e parceiros citados:

- Itaú;
- Banco do Brasil;
- Bradesco;
- Santander;
- Caixa;
- Consórcio Servopa;
- Porto Seguro;
- Embracon.

A apresentação desses bancos deve ser elegante.

Evitar parecer uma lista solta.

Criar uma faixa premium, seção de confiança ou grid refinado.

--------------------------------------------------

# GRÁFICOS

Gráficos podem ser usados, mas devem ser educativos e conceituais.

Nunca inventar números reais da Clarice.

Usar gráficos como representação visual de conceitos como:

- planejamento financeiro;
- crescimento patrimonial;
- comparação entre financiamento e consórcio;
- economia com juros;
- organização de crédito;
- evolução de patrimônio.

Sempre que necessário, incluir aviso discreto:

"Exemplo visual para fins educativos. A análise real depende de simulação personalizada."

--------------------------------------------------

# CALCULADORA OU PRÉ-SIMULAÇÃO

Se for criada uma calculadora, ela deve ser tratada como uma ferramenta de pré-simulação educativa.

Ela não deve prometer valores exatos.

Ela pode coletar escolhas como:

- objetivo do consórcio;
- tipo de bem;
- faixa de crédito desejada;
- prazo desejado;
- se mora no Brasil ou exterior.

Ao final, o CTA deve direcionar para WhatsApp com mensagem pronta.

Exemplo:

"Olá, Clarice! Quero fazer uma simulação de consórcio. Tenho interesse em imóvel e penso em uma carta na faixa de R$ 300 mil a R$ 500 mil."

--------------------------------------------------

# MOBILE

A versão mobile é prioridade.

A página deve ser impecável no celular.

Regras:

- hero bem adaptada;
- textos curtos;
- botões grandes e fáceis de clicar;
- imagens bem posicionadas;
- cards empilhados com respiro;
- nada espremido;
- nenhuma seção quebrada;
- CTA sempre claro;
- navegação simples;
- performance boa.

Não aceitar uma página bonita apenas no desktop.

--------------------------------------------------

# SENSAÇÃO FINAL DA PÁGINA

Ao terminar de navegar, o visitante deve sentir:

"Clarice Margon é uma especialista séria."

"Ela entende de consórcio, patrimônio e planejamento."

"Essa página parece confiável."

"Eu posso chamar ela no WhatsApp para fazer uma simulação."

"Isso não parece venda forçada. Parece orientação estratégica."
