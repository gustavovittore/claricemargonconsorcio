# REFERENCES — CLARICE MARGON CONSÓRCIOS

## OBJETIVO DESTE DOCUMENTO

Este arquivo reúne as referências estratégicas, visuais e estruturais para orientar a reconstrução da landing page de Clarice Margon.

A primeira versão criada ficou genérica, com copy fraca, estrutura pouco comercial e estética distante da referência desejada.

A nova versão deve usar a landing page do Ezequiel do Consórcio como principal referência de:

- estrutura;
- ritmo visual;
- diagramação;
- copy comercial;
- números animados;
- seções de serviços;
- jornada do consórcio;
- mapa mundi;
- prova social;
- seção sobre;
- conversão para WhatsApp.

Importante:

As referências devem orientar a composição e a experiência, mas NÃO devem ser copiadas literalmente.

A landing page da Clarice deve ter identidade própria, com estética mais elegante, feminina sofisticada, clean, premium e consultiva.
